# Compile
For linux:
```
git clone https://github.com/0xkhch/rectcut.git &&
cd rectcut &&
make && ./build/main
```

For windows:
```
git clone https://github.com/0xkhch/rectcut.git &&
cd rectcut &&
make && .\build\main.exe
```
